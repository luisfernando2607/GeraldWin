# 🍀 GeraldWin

## La Heralda de la Suerte tiene nombre

Bienvenido a **GeraldWin**, una aplicación web interactiva diseñada para brindar una experiencia moderna, sencilla y accesible para la selección de números y sorteos digitales.

GeraldWin combina una interfaz amigable, elementos visuales atractivos y una experiencia optimizada para dispositivos móviles, permitiendo acceder fácilmente mediante enlaces compartidos por WhatsApp o redes sociales.

## 📦 Instalación

```bash
git clone https://github.com/TU-USUARIO/geraldwin.git

cd geraldwin

npm install

npm run dev
```

---

## 🏗️ Compilar para producción

```bash

```
